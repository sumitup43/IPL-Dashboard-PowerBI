# 🏏 IPL Analysis Dashboard (2008–2025)

This project is an interactive **IPL Analysis Dashboard** developed in **Power BI** to analyze Indian Premier League (IPL) data from **2008 to 2025**.

The dashboard provides season-wise insights into team performance, player achievements, match statistics, and tournament results. Users can select a specific season and explore detailed IPL statistics through interactive visualizations.

## 🎯 Key Features
### Season Summary
- Season Winner
- Runner-Up Team
- Total Matches Played
- Total Teams Participated
- Total Venues

### Batting Analysis
- Orange Cap Holder
- Total Runs by Orange Cap Holder
- Maximum 4's
- Maximum 6's
- Total 4's in Season
- Total 6's in Season
- Total Centuries
- Total Half-Centuries

### Bowling Analysis
- Purple Cap Holder
- Total Wickets by Purple Cap Holder

### Team Analysis
- Complete Points Table
- Matches Played
- Matches Won
- Matches Lost
- Total Points

### Interactive Features
- Season Filter (2008–2025)
- Dynamic KPI Cards
- Team Rankings
- Player Performance Analysis

## 🛠️ Tools & Technologies

- Power BI
- DAX
- Power Query
- Excel / CSV Dataset
